# Rocket League Mechanics

> Visualize all mechanics with a nice table + video clips at: **https://0byte-coding.github.io/rocket_league_mechanics**

This repository contains a comprehensive, up-to-date collection of **Rocket League mechanics** — from fundamentals to advanced tech.

The mechanics data lives in [packages/rl_mechanics_data/asset/data.jsonc](packages/rl_mechanics_data/asset/data.jsonc). Each entry includes name, description, difficulty, viability, tags, video clips, and more.

---

Special thanks to [u/Accurate_Echidna_216](https://www.reddit.com/user/Accurate_Echidna_216/) for the initial provided mechanics list for the game. This project wouldn't be possible without their contributions.

---

Contributions are very welcome! If a mechanic is missing, incorrect, or outdated, feel free to open an issue or submit a PR to adjust the list in [data.jsonc](packages/rl_mechanics_data/asset/data.jsonc)
